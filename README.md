# 🌍 Système d'Analyse de Catastrophes Naturelles

Application d'analyse d'images pour la détection de catastrophes naturelles (incendies, inondations, fumée) avec estimation de gravité et montant d'indemnisation.

## 🏗️ Architecture

- **Spring Boot 3.2** - Backend (port 8080)
- **FastAPI + YOLOv8** - Service IA (port 8000)
- **H2 Database** - Stockage local

## 📋 Prérequis

- Java 17+
- Python 3.10+
- Maven (ou wrapper inclus)
